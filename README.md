### Install Prerequisites

    sudo apt-get install git openjdk-8-jdk openjdk-8-source

### Compute all solutions to 8 queens puzzle in java

Run with:

    ./gradlew run

or build and run a jar with:

    ./gradlew jar
    java -jar build/libs/queens.jar
